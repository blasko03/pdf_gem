## Usage
How to use my plugin.

## Prerequisites
This package depends on node.js and puppeteer run this command:

```bash
$ npm install -g node puppeteer
```

## Installation
Add this line to your application's Gemfile:

```ruby
gem 'pdf_gem'
```

And then execute:
```bash
$ bundle install
```

Or install it yourself as:
```bash
$ gem install pdf_gem
```
